Add text alignment controls to Elementor form fields.

== Description ==
Form Text Alignment for Elementor adds text alignment controls to your Elementor form fields, giving you more control over how your form content is displayed.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/form-alignment-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the new alignment controls in your Elementor form widgets.

== Frequently Asked Questions ==
= Does this work with all Elementor forms? =
Yes, this plugin works with all Elementor Pro form.
